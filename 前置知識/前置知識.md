- 一個數加 0 會等於原本的數；0 加一個數也會等於原本的數。
  - $2 + 0 = 2$
  - $0 + 7 = 7$

- 一個數減掉他自己，會等於 0；一個數減掉 0，會等於他自己。
  - $5 - 5 = 0$
  - $7 - 0 = 7$
