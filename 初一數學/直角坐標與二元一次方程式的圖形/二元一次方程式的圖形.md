# 二元一次方程式的圖形

- 二元一次方程式解的圖形： $ax + by = c$
	- 在坐標平面上，二元一次方程式的圖形是一條直線。
	- 繪製二元一次方程式解的圖形
		- 二元一次方程式組的解有無限多組，可用表格方式記錄。
		- 先任意給 x 的數值，再算出 y 的數值，也可以反過來。
		- 將解寫成數對的形式，描繪至坐標平面上就是一個點，最少兩個點便能連成一條直線。 
	- 找直線與兩軸的交點
		- 代入 x = 0 得到的解，便是交於 y 軸的點。
		- 代入 y = 0 得到的解，便是交於 x 軸的點。
	- $ax + by = 0 \to 直線一定會通過原點$
