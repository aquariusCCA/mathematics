# 二元一次式
- 同類項 : 文字符號相同、次方也相同的就叫做 **同類項** 。
	- $2x 、 5x$
	- $4y 、 -5y$

- 常數項 : 沒有符號，只有數字。

- 分配律 : $a(b+c+d)=a*b+a*c+a*d$

# 二元一次方程式

- 二元一次方程式 : 一個有兩種未知數，而且兩種未知數的次方都是一次的等式，就叫做 **二元一次方程式** 。
- $2x+y=10$
