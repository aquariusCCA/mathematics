# 以文字符號列式
- 代數 : 以符號 **代** 表的 **數** 。

- 由 **數字** 和 **符號** 組成的算式，稱之為 **代數式** 。
	- $20 * x$

- 代數式的簡記
	- $3 * x = 3‧x=3x$

# 求代數式的值
- 可以用帶有未知數的式子來表示一個值。
- 當未知數的值不同的時候，式子的值也會不同。
