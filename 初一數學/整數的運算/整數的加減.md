# 整數的加法運算
- 同號數相加 : 兩個同號數相加時，其結果等於兩數的絕對值相加，而性質符號與原來兩數相同。
- 異號數相加 : 兩個異號數相加時，其結果等於兩數的絕對值相減 (大減小)，而性質符號與絕對值較大的數相同。
