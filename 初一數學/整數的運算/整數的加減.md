# 整數的加法運算
- 同號數相加 : 兩個同號數相加時，其結果等於兩數的絕對值相加，而性質符號與原來兩數相同。
- 異號數相加 : 兩個異號數相加時，其結果等於兩數的絕對值相減 (大減小)，而性質符號與絕對值較大的數相同。

# 數線上兩點的距離
- 數線上兩點間的距離 : 一數線上有 A(a)、B(b) 兩點，則 A、B 兩點間的距離 $\overline{AB}=|a-b|=|b-a|$ ，也就是 a、b 兩數中，大數減小數的值。
