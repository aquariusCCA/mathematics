# 整數的乘法運算
- 整數的乘法 :
	- 同號的兩整數相乘，其結果為 **正整數** 。
	- 異號的兩整數相乘，其結果為 **負整數** 。

- 在計算整數的乘法時
	- 先判斷正負的性質
		- 正正得正
		- 負正得負
		- 正負得負
	- 計算數字部分的乘法

- 負數乘以負數會得到正數，也就是 **負負得正** 。
	- ex : $(-2)*(-5)=10$

- 有一個數 a
	- $a*1=a$
	- $a*0=0$
	- $a*(-1)=-a$

- 乘法運算規則 :
	- 交換律 : $a*b=b*a$
	- 結合律 : $(a*b)*c=a*(b*c)$
	- 分配律 : $(a+b)*c=a*c+b*c$
